# Security Policy

## Supported Versions

The current public development version is supported for security reports.

## Reporting A Vulnerability

Please report security concerns privately through YASA LTD:

- Contact: https://yasa.fi/contact/
- Email: info@yasa.fi

Do not open a public GitHub issue for exploitable vulnerabilities. Include reproduction steps, affected versions, and any relevant logs or proof of concept details.

## Data Handling

OpenConsent CMP is designed to keep consent settings and anonymized consent logs inside the WordPress installation. It should not send consent records to YASA LTD or any external service.
