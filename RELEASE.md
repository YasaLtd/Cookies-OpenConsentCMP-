# Release Checklist

1. Update `Version` in `openconsent-cmp.php`.
2. Update `OPENCONSENT_CMP_VERSION`.
3. Update `Stable tag` and changelog in `readme.txt`.
4. Refresh `languages/openconsent-cmp.pot` if strings changed.
5. Rebuild `openconsent-cmp.zip`.
6. Copy the ZIP into `cookies-yasa-site/downloads/openconsent-cmp.zip`.
7. Verify the live website demo and plugin frontend controller match.
8. Tag the GitHub release.

Suggested tag format: `v1.0.3`.
